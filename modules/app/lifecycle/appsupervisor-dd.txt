= AppSupervisor

== Description

I am the top-level application supervisor.
I own startup and shutdown wiring and I provide the primary C++ surface presented to QML.

I collaborate with UI-facing adaptors and service modules by owning their lifetimes and wiring them together.
I do not implement capture, networking, rendering, or HTTP export myself; I orchestrate those services.

State model:
- Thread affinity: UI thread.
- Lifetime: constructed at program startup and destroyed at process shutdown.

== Public interface

- `QString appVersion() const` returns the application version string.

- `void shutdown()` performs orderly shutdown of owned services.
  - It currently persists preferences.

- `bool joinRoomEnabled() const` returns whether the local instance is configured to join a room.
- `void setJoinRoomEnabled( bool enabled )` sets whether the local instance is configured to join a room.

  Notes:
  - Setting the property to its current value is a no-op and does not emit a change signal.

- `bool hostRoomEnabled() const` returns whether the local instance is configured to host a room.
- `void setHostRoomEnabled( bool enabled )` sets whether the local instance is configured to host a room.

  Notes:
  - Setting the property to its current value is a no-op and does not emit a change signal.

- `QObject *preferences() const` returns the application preferences object.
- `QObject *deviceCatalogue() const` returns the local device catalogue object.
- `QObject *fontPreviewSafetyCache() const` returns the font preview safety cache.

  Ownership:
  - Returned pointers are owned by `AppSupervisor` (QObject parented to it) and remain valid for the supervisor's lifetime.

- `bool themeIconAvailable( const QString &name ) const` returns whether a named theme icon is available.

- `QStringList fontFamilies() const` returns available font family names installed on the system.
  - Intended for populating QML font selection UI.

- `QString systemFontFamily() const` returns the platform's general system UI font family.
  - Intended for rendering fallback UI copy when the user-selected app font renders incorrectly.

- `QObject *fontPreviewSafetyCache() const` returns a UI-facing cache that can check whether a font can be preview-rendered safely.
  - Intended for use by the font picker to avoid repeatedly attempting previews that are known to render incorrectly on this machine.
  - Non-goal: this cache must not prevent the user selecting a font; it is only advisory.

Signals:
- `joinRoomEnabledChanged()` is emitted when `joinRoomEnabled` changes.
- `hostRoomEnabledChanged()` is emitted when `hostRoomEnabled` changes.

Thread safety:
- Thread-safe: No.
- Re-entrant: No.

Errors:
- This module currently has no failure paths.

== Implementation details

- `appVersion()` is currently sourced from `QCoreApplication::applicationVersion()`.
- `shutdown()` is currently a placeholder and will gain real behaviour as services are introduced.
  It currently persists preferences as part of orderly shutdown.

== Requirements

- Language: C++17.
- Qt: 6.x.
