 = FontPreviewSafetyCache

 == Description

 I am a UI-facing cache that checks whether a font can be preview-rendered safely.
 I help the application detect fonts that render incorrectly on the current machine so UI preview code can avoid repeatedly producing corrupted preview output.

 I collaborate with QML font picker UI code that renders per-row previews.
 I collaborate with `FontPreviewSafetyChecker` to perform font preview checks.
 I collaborate with the default checker implementation `FontPreviewSafetyCheckerDefault`.
 I do not attempt to decide which fonts the user is allowed to select.
 I do not guarantee that a font is safe for all use cases; I only provide a best-effort signal for the application's preview workload.

 State model:
 - Thread affinity: the public API is UI-thread only.
 - Lifetime/ownership: owned by a UI supervisor (for example `AppSupervisor`) and lives for the application lifetime.
 - Re-entrancy: not re-entrant.
 - Concurrency: each check runs asynchronously via a checker and reports results back to the UI thread.
 - Timeouts: checks that do not complete within a time budget are marked as `TimedOut`.
 - Safety: timed-out checks are treated as potentially stuck inside the font stack and are not forcibly terminated.
 - Backpressure: once a fixed number of checks are timed out, new checks are marked `Blocked` to prevent unbounded resource growth.

 == Public interface

 Types:
 - `enum Status { NeverChecked, Checking, Safe, Bad, TimedOut, Blocked }`
   - `NeverChecked`: no check has been started yet.
   - `Checking`: a check is in progress.
   - `Safe`: the check indicates the preview output is consistent with a reference shape.
   - `Bad`: the check indicates the preview output is inconsistent with a reference shape.
   - `TimedOut`: the check did not complete within the time budget; the result is inconclusive.
   - `Blocked`: a check was requested but was not started because too many previous checks timed out.

 Functions:
 - `Status statusForFamily( const QString &family, int pixelSize ) const`
   - Returns the cached status for the requested font family at the given pixel size.
   - If no status exists yet, returns `NeverChecked`.

 - `void requestCheck( const QString &family, int pixelSize )`
   - Starts an asynchronous font preview check if the status is currently `NeverChecked` or `Blocked`.
   - If the status is already `Checking`, `Safe`, `Bad`, or `TimedOut`, it is a no-op.
   - If too many previous checks are timed out (likely stuck), `requestCheck` sets the status to `Blocked`.
   - `Blocked` entries are eligible to be checked again when `requestCheck` is called in the future and the cache has capacity.

 Constructors:
 - `FontPreviewSafetyCache( QObject *parent = nullptr )` constructs a cache with the default checker implementation.
 - `FontPreviewSafetyCache( FontPreviewSafetyChecker *checker, const Config &config, QObject *parent = nullptr )` constructs a cache with an injected checker.

 Signals:
 - `statusChanged( const QString &family, int pixelSize, Status status )`
   - Emitted when a cached status changes.
   - Consumers should treat the result as advisory and should not block user actions.

 Thread safety:
 - Thread-safe: No.
 - Calls must be made from the UI thread.

 Errors:
 - This cache does not throw or report hard errors.
 - Fonts that are missing or that resolve to a fallback are treated as `Safe` for preview purposes.

 == Implementation details

 - By default, the cache constructs `FontPreviewSafetyCheckerDefault`.
 - The cache can also be constructed with an injected checker (used by deterministic unit tests).
 - Checks are time-boxed; timeouts are inconclusive and yield `TimedOut`.
 - To avoid unbounded resource growth in the presence of stuck font stack calls, the cache stops starting new checks after a fixed number of timed-out checks, marking new requests as `Blocked`.
   - The default limit is a single constant in source (`VCSTREAM_FONT_PREVIEW_SAFETY_DEFAULT_MAX_STUCK_CHECKS`).
 - Results are cached by `(family, pixelSize)`.

 == Requirements

 - Language: C++17.
 - Qt: 6.x (QtCore, QtGui).
